# Illia Filimonov — Portfolio (Liquid Glass)

Готовый исходник портфолио в стиле Apple Liquid Glass. Двуязычный (RU/EN, переключатель в шапке).

## Как открыть
Файлы нужно отдавать через локальный веб-сервер (не по `file://`, иначе не подгрузятся шрифты/картинки).
Пример:
```
cd этой_папки
python3 -m http.server 8000
# открыть http://localhost:8000/Illia%20Filimonov%20Portfolio.dc.html
```

## Структура
- `Illia Filimonov Portfolio.dc.html` — главный файл. Внутри:
  - первый `<style>` — @font-face (локальные шрифты) + подключение рантайма;
  - `<helmet><style>` — вся дизайн-система (стекло, цвета, сетки, компоненты);
  - `<x-dc> … </x-dc>` — разметка страницы (секции, блоки);
  - `<script type="text/x-dc">` в конце — данные и логика (весь текст RU/EN, массивы опыта, кейсов, клиентов, переключатель языка).
- `support.js` — рантайм, который оживляет `.dc.html`. Держать рядом с html.
- `assets/` — изображения:
  - `img1.jpg` — фото, `img2.png` — Google Partner, `img3.png` — лого Sarafunka, `img4.png` — дашборд Google Ads;
  - `cert-*-badge.png` — бейджи сертификаций;
  - `logos/*.png` — логотипы клиентов.
- `fonts/` — веб-шрифты: Manrope (вариативный, латиница+кириллица) + JetBrains Mono, woff2.

## Где что править
- **Тексты / данные** (заголовки, абзацы, опыт, кейсы, клиенты) — в блоке `<script type="text/x-dc">`, объект `T` (ключи `ru` / `en`).
- **Стили / стекло / цвета** — в `<helmet><style>`. Основные токены в `:root` (`--coral`, `--frost`, `--bd`, `--ao` и т.д.).
- **Разметка / порядок секций** — в `<x-dc>`.
